# 🎓 Elead School - Premium Landing Page

Une landing page ultra-moderne, interactive et multilingue conçue pour **Elead School** (École Maternelle et Primaire à Tanger). Ce projet a été développé avec une attention particulière portée à l'expérience utilisateur (UX), aux animations fluides et à l'adaptabilité mobile.

## ✨ Fonctionnalités Principales

* 🌍 **Multilingue (i18n) :** Support complet du Français, de l'Anglais et de l'Arabe, incluant une gestion parfaite de la direction du texte (RTL) pour l'arabe.
* 🌓 **Mode Sombre / Clair :** Thème dynamique (Light/Dark mode) intégré nativement.
* ✨ **Animations Premium :** Effets d'apparition au défilement (scroll) et micro-interactions développés avec *Framer Motion*.
* 📱 **Design 100% Responsive :** Approche "Mobile-First" avec des grilles adaptées et une navigation fluide sur tous les appareils.
* 🖼️ **Galerie Interactive (Lightbox) :** Visualisation des images en plein écran avec un système de modale personnalisé.
* ⭐ **Page "Superstars" Dédiée :** Système de routage pour afficher la liste complète des élèves méritants sur une page séparée.
* 🚀 **SEO Optimisé :** Balises sémantiques, balises meta dynamiques et structure HTML pensée pour le référencement naturel.

## 🛠️ Stack Technique

* **Framework :** React.js (via Vite)
* **Langage :** TypeScript (TSX)
* **Styling :** Tailwind CSS
* **Animations :** Framer Motion
* **Routage :** React Router DOM
* **Icônes :** Lucide React
* **Drapeaux :** FlagCDN (optimisé pour tous les OS)

## 🚀 Installation et Lancement

Pour lancer ce projet localement, suivez ces étapes :

1. **Cloner le dépôt :**
   ```bash
   git clone [https://github.com/votre-nom-d-utilisateur/elead-school-landing.git](https://github.com/votre-nom-d-utilisateur/elead-school-landing.git)
Accéder au dossier du projet :

Bash
cd elead-school-landing
Installer les dépendances :

Bash
npm install
Lancer le serveur de développement :

Bash
npm run dev
Le projet sera accessible sur http://localhost:5173 (ou le port indiqué dans votre terminal).
